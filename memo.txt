アップデート用
npm install zenn-cli@latest

以下のコマンドによりmarkdownファイルを簡単に作成できます。
npx zenn new:article
slugを指定する場合（以下の場合は　what-is-slug　になる）
npx zenn new:article --slug what-is-slug

ブラウザでプレビューするためには次のコマンドを実行します。
npx zenn preview

記事の追加方法

git add .

git commit -m "新記事追加"

(-m "新記事追加"：「-m」はGit内部にコミットメッセージが保存されます。)

git push
